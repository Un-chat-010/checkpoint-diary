# 检查点 · Checkpoint

> 一个单文件 HTML 深度思考工具，集成了 Ali Abdaal 推荐的四个自我反思框架。打开 → 说说你现在的状态 → AI 判断你需要什么 → 进入对应工具 → 生成可视化报告。

---

## 四个工具

| 工具 | 适合时机 |
|------|----------|
| 🌑 **恐惧设定** | 有件事一直在拖，迟迟不动 |
| 🪞 **所罗门悖论** | 在某件事上卡住了，需要换个视角 |
| 🗺️ **奥德赛计划** | 感到迷茫，不知道自己想要什么方向 |
| ⭕ **生命之轮** | 状态模糊，需要先摸清楚哪个维度出了问题 |


## 设计考量

为 AuDHD 使用习惯设计：

- 单屏单任务，无需在多个页面间跳转
- 打开即开始，不需要先"选择今天要做什么"
- 标签点选降低文字输入门槛
- 报告有视觉锚点，不依赖大段阅读
